# Employee Productivity Analysis (R)

## Overview
This project analyzes factors affecting employee productivity using regression models.

## Methods
- Log transformation
- Regression analysis
- Outlier detection
- Model diagnostics (VIF, Breusch-Pagan, Durbin-Watson)

## Key Findings
- Experience and salary increase productivity
- Overtime decreases productivity (burnout effect)
- Model explains ~97% of variance
